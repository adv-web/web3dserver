Class: GameServer

Defined in: src\game\GameServer.coffee

Overview

This class handle the message about players create a game, players join a game, players leave a game and host start a game and so on.

It will set up a game according to the request to the user.

It's namespace is /game

Examples:

# if the host of the game want to add some tree in the game
game.registerComponent(new TreeLoader(player))
